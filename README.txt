# 🌙 MoodMirror – Your Mind in One Screen

MoodMirror is a simple yet powerful AI-inspired web application that reflects your emotional state based on what you type.

Instead of just analyzing text, it gives you:

* 🧠 A mood classification
* 🎬 A cinematic reflection line
* ⚡ A small actionable suggestion

---

## 🚀 Demo

Type how you feel → Get an instant emotional reflection.

Example:

> "I feel tired and lost"
> → Mood: Low
> → "Even the darkest nights end with sunrise."
> → Suggestion: Take a short walk or listen to your favorite song.

---

## 💡 Why This Project?

Most student projects focus only on functionality.
MoodMirror focuses on **experience + emotion + simplicity**.

This project is designed to:

* Feel like a **product**, not an assignment
* Be **visually engaging** for LinkedIn demos
* Show **creative thinking + UX understanding**

---

## 🧠 How It Works

* Uses simple keyword-based sentiment detection
* Maps emotions into:

  * Low
  * Neutral
  * High
* Returns:

  * A cinematic reflection line
  * A practical suggestion

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS

## ⚙️ Installation & Setup

### 1. Clone the repository

```
git clone https://github.com/your-username/moodmirror.git
cd moodmirror
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the application

```
python app.py
```

### 4. Open in browser

```
http://127.0.0.1:5000
```
